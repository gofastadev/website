# Feature Flags

The `featureflag` package provides feature flag evaluation using [go-feature-flag](https://github.com/thomaspoignant/go-feature-flag). It supports file-based (YAML) flag providers with periodic polling, user-targeted evaluation, and boolean flag variations.

## Import

```go
import "github.com/gofastadev/gofasta/pkg/featureflag"
```

## Key Types

### FeatureFlagService

```go
type FeatureFlagService struct {
    logger *slog.Logger
}
```

## Key Functions

| Function | Signature | Description |
|----------|-----------|-------------|
| `NewFeatureFlagService` | `func NewFeatureFlagService(configPath string, logger *slog.Logger) (*FeatureFlagService, error)` | Initializes the feature flag system from a YAML config file |
| `IsEnabled` | `func (s *FeatureFlagService) IsEnabled(flagKey string, userID string, attributes map[string]interface{}) bool` | Checks if a feature flag is enabled for a given user context |
| `Close` | `func (s *FeatureFlagService) Close()` | Shuts down the feature flag client |

## Flag Definition File

Flags are defined in a YAML file that follows the [go-feature-flag format](https://gofeatureflag.org/docs/):

`flags.yaml`:

```yaml
new-dashboard:
  variations:
    enabled: true
    disabled: false
  targeting:
    - query: key eq "user-123"
      variation: enabled
  defaultRule:
    percentage:
      enabled: 50
      disabled: 50

dark-mode:
  variations:
    enabled: true
    disabled: false
  defaultRule:
    variation: enabled
```

## Usage

### Initializing the Service

```go
ffService, err := featureflag.NewFeatureFlagService("flags.yaml", logger)
if err != nil {
    log.Fatalf("failed to load flags: %v", err)
}
defer ffService.Close()
```

### Basic Flag Checking

```go
if ffService.IsEnabled("dark-mode", userID, nil) {
    // serve dark mode UI
}
```

### User-Targeted Flags with Attributes

```go
attributes := map[string]interface{}{
    "plan":    "premium",
    "country": "US",
}

if ffService.IsEnabled("new-dashboard", userID, attributes) {
    // show new dashboard
}
```

### Using in Controllers

```go
func (c *DashboardController) Index(w http.ResponseWriter, r *http.Request) {
    claims := auth.ClaimsFromContext(r.Context())

    if c.ffService.IsEnabled("new-dashboard", claims.UserID, nil) {
        httputil.JSON(w, http.StatusOK, newDashboardData)
        return
    }
    httputil.JSON(w, http.StatusOK, legacyDashboardData)
}
```

### Configuration via config.yaml

```yaml
feature_flag:
  enabled: true
  config_path: flags.yaml
```

Environment variables use the `GOFASTA_` prefix (e.g., `GOFASTA_FEATURE_FLAG_ENABLED`, `GOFASTA_FEATURE_FLAG_CONFIG_PATH`).
