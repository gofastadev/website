# gofasta g route

Generates a route definition file that maps HTTP endpoints to controller handlers using gorilla/mux. The generated routes follow RESTful conventions with standard CRUD paths.

## Usage

```bash
gofasta g route <Name>
```

This command has no flags.

## Examples

Generate routes for a Product resource:

```bash
gofasta g route Product
```

## What It Generates

Running `gofasta g route Product` creates one file:

```
app/rest/routes/product.routes.go
```

### Generated Code

```go
// app/rest/routes/product.routes.go
package routes

import (
    "github.com/gorilla/mux"
    "myapp/app/rest/controllers"
    "myapp/pkg/httputil"
)

func RegisterProductRoutes(router *mux.Router, controller *controllers.ProductController) {
    sub := router.PathPrefix("/api/v1/products").Subrouter()

    sub.HandleFunc("", httputil.Handle(controller.Create)).Methods("POST")
    sub.HandleFunc("", httputil.Handle(controller.FindAll)).Methods("GET")
    sub.HandleFunc("/{id}", httputil.Handle(controller.FindByID)).Methods("GET")
    sub.HandleFunc("/{id}", httputil.Handle(controller.Update)).Methods("PUT")
    sub.HandleFunc("/{id}", httputil.Handle(controller.Delete)).Methods("DELETE")
}
```

## Route Registration

The generated route file is registered in `app/rest/routes/index.routes.go`. When using `gofasta g scaffold`, this registration is patched automatically. When generating routes standalone, you need to add the registration manually:

```go
// app/rest/routes/index.routes.go
func RegisterRoutes(router *mux.Router, container *di.Container) {
    RegisterUserRoutes(router, container.UserController)
    RegisterAuthRoutes(router, container.AuthController)
    RegisterProductRoutes(router, container.ProductController) // Add this line
}
```

## Generated Endpoints

| Method | Path | Handler |
|--------|------|---------|
| `POST` | `/api/v1/products` | `Create` |
| `GET` | `/api/v1/products` | `FindAll` |
| `GET` | `/api/v1/products/{id}` | `FindByID` |
| `PUT` | `/api/v1/products/{id}` | `Update` |
| `DELETE` | `/api/v1/products/{id}` | `Delete` |
