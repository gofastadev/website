# gofasta version

Displays the CLI version, Go version, and OS/architecture details.

## Usage

```bash
gofasta version
```

This command takes no flags. You can also use the `-v` or `--version` flag on the root command:

```bash
gofasta -v
```

## Examples

```bash
$ gofasta version
gofasta version 0.6.0
Go:     go1.25.8
OS:     darwin/arm64
```
